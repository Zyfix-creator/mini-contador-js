# Mini Contador JS

## Descrição
Um projeto simples em **JavaScript** que permite ao usuário contar números de forma **crescente ou decrescente**.  
Ao final, o programa classifica a contagem como **pequena**, **média** ou **grande**.

Este projeto foi desenvolvido para praticar:

- Laços de repetição (`for`, `while`)
- Condicionais (`if / else if / else`, `switch`)
- Entrada de dados do usuário
- Variáveis e operadores básicos

---

## Funcionalidades

1. Pergunta ao usuário um número limite da contagem.
2. Pergunta se quer contar de forma crescente ou decrescente.
3. Mostra a contagem no console.
4. Classifica o número informado:
   - < 5 → Contagem pequena
   - 5 a 9 → Contagem média
   - ≥ 10 → Contagem grande
